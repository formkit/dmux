Mission Control for CLI Agents

Run Claude Code or opencode in parallel tmux panes with git worktrees

$ npm install -g dmux View on GitHub →
~/projects/my-app/main
$ dmux 

          

        

      

    

  


  
  

    

      

        $ Why dmux?
      

      

        

          

          

          

            
          

          
Parallel Development

          
Work on multiple features simultaneously. Each pane is a separate tmux pane with its own git worktree.

          $ dmux
# Press 'n' → Enter prompt → Agent launches
        

        

          

          

          

            
          

          
Git Worktree Isolation

          
Each pane operates in its own git worktree. No more branch switching or stashing changes.

          .dmux/worktrees/
  ├── fix-auth/
  └── add-dashboard/
        

        

          

          

          

            
          

          
AI-Powered Automation

          
Automatic branch naming and commit message generation using AI. Smart merge workflows.

          "fix authentication bug"
→ Branch: fix-auth
→ Commit: "fix: resolve auth token..."
        

        

          

          

          

            
          

          
Agent Ready

          
Launches Claude Code (with auto-accept edits) or opencode with your prompt. Ready to work instantly.

          Claude: --permission-mode=acceptEdits
opencode: prompt auto-submitted
        

      

    

  


  
  

    

      

        $ Installation
      

      

        

          
1

          

            
Install dmux

            

              
              
$ npm install -g dmux

            

          

        

        

          
2

          

            
Install an Agent

            

              
              
# Install Claude Code or opencode
$ npm install -g claude-code

        $ How It Works
      

      

        

          

            
          

          
1. Start dmux

          
Run dmux in your project directory

          

            
$ cd ~/projects/my-app/main
$ dmux

          

        

        
→

        

          

            
          

          
2. Create Panes

          
Press n and enter your prompt

          

            
Enter prompt: fix authentication bug

# Creates worktree, launches agent

          

        

        
→

        

          

            
          

          
3. Merge Work

          
Press m when done to merge and clean up

          

            
# Auto-commits, merges to main
$ git log -1
fix: resolve authentication token expiry

          

        

      

    

  


  
  

    

      

        $ Keyboard Shortcuts
      

      

        

          n
          Create new pane
        

        

          j or Enter
          Jump to pane
        

        

          m
          Merge to main
        

        

          x
          Close pane
        

        

          ↑ ↓
          Navigate list
        

        

          q
          Quit dmux
        

        

          Esc
          Cancel dialog
        

      

    

  


  
  

    

      

        $ Perfect For
      

      

        

          
Bug Hunting

          
Spin up multiple panes to test different fixes in parallel. Keep the winner, close the rest.

        

        

          
Feature Development

          
Work on the main feature while agents handle tests, docs, and refactoring in separate panes.

        

        

          
Experimentation

          
Try different approaches without commitment. Each worktree is isolated and disposable.

        

        

          
Team Workflows

          
Share dmux sessions for pair programming with AI agents alongside human developers.
